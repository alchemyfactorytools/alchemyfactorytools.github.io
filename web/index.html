Capacity & capital

How much copper per item you'll waste for a simpler factory. It finds the cheapest build, then within that budget picks the one with the fewest distinct production lines — always a single clean route per item, never a split. Budget scales to the build's own cheapest per-item cost (same for a 2g 80s Mars or a 1c nail). "Simplest" ignores cost entirely and takes the globally fewest-line build (e.g. buy ore → smelter over a sprawling free-herb cauldron chain).

Set an exact per-item waste budget instead of the dropdown's scaled default. e.g. set it to 5 and a 1c-vs-4c-per-nail difference stops mattering — it'll pick a one-machine ore smelter over a free-herb cauldron chain.

Counts what each machine costs to build (including the Nursery plots a low-quality fertilizer implies), so the planner won't pick a thousand-nursery sprawl just because the materials are cheap. Use Optimize for above to trade factory size against running cost.
